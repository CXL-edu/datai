# 表格数据

**tabulate**是一个用于在命令行/终端中创建格式化表格的Python库

tabulate主要功能：

- 将数据以各种表格格式输出
- 支持多种数据源(列表、字典、pandas DataFrame等)
- 支持多种表格样式(grid, pipe, orgtbl, markdown等)

```python title="将表格数据保存为markdown表格格式"
from tabulate import tabulate

data = [
    ["John", 30, "New York"],
    ["Alice", 25, "Boston"],
    ["Bob", 35, "Chicago"]
]
headers = ["Name", "Age", "City"]

# 输出markdown格式
markdown_table = tabulate(data, headers=headers, tablefmt="pipe")
print(markdown_table)

# 保存到文件
with open('output.md', 'w', encoding='utf-8') as f:
    f.write(markdown_table)
```


- tabulate支持多种tablefmt选项，以下是常用的格式：
  - 1、简单格式：
    - `simple` - 简单的ASCII表格
    - `plain` - 简单的空格分隔表格
    - `grid` - ASCII网格表格
    - `pretty` - 漂亮的ASCII表格
  - 2、Markdown相关：
    - `pipe` - Markdown格式表格
    - `github` - GitHub风格Markdown
    - `jira` - Jira wiki格式
  - 其他格式：
    - `rst` - reStructuredText格式
    - `html` - HTML表格
    - `latex` - LaTeX表格
    - `latex_raw` - LaTeX表格（无格式）
    - `latex_booktabs` - LaTeX booktabs格式
    - `tsv` - Tab分隔值
    - `orgtbl` - Org-mode表格
    - `mediawiki` - MediaWiki表格格式
    - `moinmoin` - MoinMoin wiki表格格式
    - `textile` - Textile格式
    - `presto` - CSV格式（以逗号分隔）
