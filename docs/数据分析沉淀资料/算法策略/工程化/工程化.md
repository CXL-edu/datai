# 工程化

先用serverless服务

### （1）批量数据处理 / 特征工程

这里确实有分布式需求。推荐：

- **Spark**（比 Hadoop MapReduce 更灵活，支持 SQL、MLlib、流处理，Python API 友好）。
- **Dask / Ray**（Python 原生，支持 Pandas / Numpy / Scikit-learn 的分布式版本）。
- 好处：你写的 Python 代码只需小改动，就能扩展到多核/多机。

### （2）模型训练 / 推理（需要 GPU）

- MapReduce 不擅长 GPU 调度。
- 更合适的方案是**分布式深度学习框架**：PyTorch Distributed、TensorFlow Distributed、Horovod。
- 或者用**Ray**（支持同时调度 CPU + GPU 任务）。
