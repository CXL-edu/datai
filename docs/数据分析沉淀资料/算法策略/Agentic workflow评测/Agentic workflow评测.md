# Agentic workflow评测

度量方法：人工评分，每项满分为2，单条数据满分为10分

度量指标：

| 动作信息完整性 | 模型总结中各操作的完整性，包含所有的必要操作                                          |
| ------- | --------------------------------------------------------------- |
| 动作顺序准确性 | 模型总结中各操的顺序是否与参考总结一致                                             |
| 对象细节准确性 | 模型总结是否明确写出了操作对象的具体细节（如App名称、筛选条件、搜索关键词、新闻标题等），而不是笼统的概括，且没有错误的细节 |
| 输出格式规范性 | 模型总结是否为一句话总结，并符合简洁直述的风格，没有冗余的解释性描述                              |
| 生成意图合理性 | 生成的意图合理且相关                                                      |

新的度量指标：

| 相关性     | 是否符合用户意图               |
| ------- | ---------------------- |
| 有用性     | 用户看到是否真的会采纳            |
| 清晰度     | 生成的文本是否简洁易懂            |
| 可执行性    | 建议是否能被用户直接执行           |
| 新颖性/惊喜感 | 建议是否能带来新的可能性，而不是仅仅重复历史 |
